Interactive Database Connected Map
This piece was developed for an advanced, Master's level course in GIS. We were tasked with developing a map application for a park, and the application could have a theme of our choosing. However, the application needed to have 5+ layers, be interactive, and enable the user to add points to the map. I accomplished this by developing a Java web app with an associated PostGIS database.

This project involved Leaflet, Java, PostGIS, Apache Tomcat, PostgreSQL, Heroku CLI, JavaScript, JQuery, and Bootstrap
